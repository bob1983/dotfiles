# dotfiles
## Usage

### Bootstrap for ubuntu
Install essential softwares and setup your rcfiles.

```
git clone https://github.com/bob1983/dotfiles ~/dotfiles
cd dotfiles
./bootstrap.sh
./setup/dev_langs.sh
./setup/symlink.sh
```

### Bootstrap for osx
Setup your rcfiles.

```
./bootstrap.sh
./setup/symlink.sh
```
